# Vue JS

## basic
### 1. SPA
Single Page Application
- HTTP 요청/응답 없이 UI 변화
- 이유
    1. 사용자 편의성
    2. 리소스

### 2. MVVM
Model-View-ViewModel
- MVC(Model-View-Control)와의 비교

### 3. Declarative Programming
선언적 프로그래밍
<-> 명령형 프로그래밍(Imperative Programing)
- 명령형(절차적) : 변화를 단계적으로 명령
    - Vanilla Javascript DOM 조작
- 선언형 : 변화를 선언(그림 그리기)
    - 프레임워크를 통한 조작
